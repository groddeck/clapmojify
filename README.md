# Clapmojify

Inject 👏 clap-emoji 👏 into 👏 your 👏 text 👏.

## Installation

To install the gem:

    $ gem install clapmojify

## Usage

At the command-line, type `clapmojify <text>` or pipe/redirect output:

    $ clapmojify fix my text

yields:

```
fix 👏 my 👏 text 👏
```

Alternately:

    $ clapmojify < input_file.txt

or

    $ cat input_file.txt | clapmojify
